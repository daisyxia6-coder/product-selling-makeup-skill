# product-selling-makeup-skill

美妆产品种草视频分镜脚本生成技能。

## 功能介绍

输入一段美妆产品上妆试色视频，自动完成：

1. 下载并提取视频音频脚本
2. 按「开场 → 功能展示 → 收尾CTA」三段式框架拆解内容
3. 产出完整分镜脚本表格（时间戳 + 场景 + 机位 + 口播 + 字幕）
4. 截取关键帧并与脚本节点对应
5. 打包 zip 交付

## 三段式框架

```
开场（0-15%）→ 功能展示（60-75%）→ 收尾CTA（15-20%）
```

### 开场钩子类型

| 类型 | 模板句式 | 适用产品 |
|------|---------|---------|
| 素颜vs完妆反差 | 「不会画xx的老婆们看过来」 | 眼线、眉笔 |
| 效果前置 | 「用它3步搞定！」 | 任何易上手产品 |
| 痛点直击 | 「xx星人必看！」 | 护肤、美白 |
| 前后对比 | 「画vs不画，差的不是一点点」 | 眼线、眉笔、高光 |
| 工具导入 | 「xx想画好，工具一定要选对」 | 眼线、眉妆 |

### 功能展示模块

1. **产品外观展示** — 特写/手持，口播类型+卖点
2. **质地测试** — 手背试色，口播触感+感受
3. **上脸演示** — 侧面近景+特写，动作词+位置描述
4. **效果对比** — 同框左右对比，大字字幕强化
5. **持久度测试** — 手指蹭/纸巾擦，场景+结果

### 收尾CTA四步

```
颜色推荐 → 套装内容 → 价格锚定 → 行动指令
```

## 字幕设计规范

| 位置 | 类型 | 示例 |
|------|------|------|
| 产品卖点 | 关键词+emoji | 「丝滑不拉扯 ✅」 |
| 动作引导 | 短句+箭头 | 「填满根部 → 眼尾后拉」 |
| 效果强调 | 大字+对比 | 画 ✅ vs ❌ 没画 |
| CTA区 | 超大字 | 「02纯咖色」「闭眼入」 |

## 安装使用

已在 OpenClaw 环境中，直接上传 `.skill` 文件即可使用，或在 Agent 对话中提供视频链接触发完整流程。

- 分镜脚本批量生成
