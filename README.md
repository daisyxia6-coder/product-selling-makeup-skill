# 爆款视频拆解和复刻
> 多种爆款视频模板类型的系统性拆解工具，覆盖美妆、食品、家居等全品类。
## 快速开始
触发词：**拆解视频、分析视频、内容拆解、爆款拆解、效果对比**
用户提供视频ID或分享链接，自动完成：
1. 下载视频
2. 提取音频脚本
3. 截取关键帧
4. 识别爆款类型
5. 输出完整分镜脚本
6. **打包交付物**：`分镜脚本 + 关键帧截图 zip`
